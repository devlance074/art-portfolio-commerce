
# 🎨 Modern Art Portfolio eCommerce WebApp

This project is a stylish, fully responsive art portfolio eCommerce web application designed for artists to both showcase and sell their artwork online. Originally created to highlight a single artist’s portfolio, it’s been enhanced to support basic eCommerce functionality — all while retaining a clean, expressive, and minimalist design.

---

## 🌐 Live Preview
You can view the live version of the project here:  
https://art-commerce.netlify.app/

## 🧠 GitHub Repository
You can access the source code via GitHub here:  
https://github.com/devlance074/art-portfolio-commerce.git

---

## 🚀 Features

- 🖼️ Home page with hero section, artist intro, and CTA
- 🖌️ Gallery grid with hover animations & artwork modal
- 📄 Detailed artwork page (title, image, description, price, etc.)
- 🛒 Basic eCommerce support: Add to cart, View cart (frontend only)
- 🙋‍♀️ About page with biography and social links
- ✉️ Contact page with a working form (Name, Email, Message)
- 🌗 Light/Dark theme toggle
- 📱 Fully responsive across all devices
- 🎨 Clean UI using off-white/beige and soft green accents (Light Mode)
- 🖤 Vibrant black and dark forest green (Dark Mode)

---

## ⚙️ Requirements

- Node.js (v16 or newer)
- npm or yarn
- A modern browser

---

## 🧩 Installation

1. **Clone the repository**

```bash
git clone https://github.com/devlance074/art-portfolio-commerce.git
```

2. **Navigate into the folder**

```bash
cd art-portfolio-ecommerce
```

3. **Install dependencies**

```bash
npm install
# or
yarn
```

4. **Start the development server**

```bash
npm run dev
# or
yarn dev
```

5. **Open in browser**

Navigate to `http://localhost:3000` to view the site.
